# 概要

生成AIを使った死にゲー。
異種族たちが暮らす世界観。

主人公は女の子ネムル。
パートナーの女の子はニーナ。
ニーナはネムルを置いて出かける。
ネムルはお留守番をするお話。
物語の冒頭、ニーナは寝ているネムルに向けて手紙を置いてから出かける。

ネムルの元にはさまざまな人が訪れる。
ネムルは勝手に人と会話し、結果的に殺されたりしてしまう。

殺されると夢オチ扱いで最初からスタートする形でループする。
プレイヤーが唯一行えるのは、このリスタートのタイミングで、ネムルへの手紙の中身をフリーテキストで書き換えること。

ネムルは、この手紙の内容を元に行動決定するので、死を回避したりすることができる。
フリーテキストを解析してネムルの行動を決定する役割を生成AIが担う。
ネムルは無事にニーナと再開することができるか。
たくさんのバッドエンドに加え、ノーマルエンドとトゥルーエンドがある。

# テーマ

みんなそれぞれの背景を抱えていて、それぞれの正義を持って行動している。
全ての人の理想が100%噛み合うことは難しくても、ほんの少しの歩み寄りや視点や行動の変化で、結末は大きく変わる。

# システム

## ヒント

ネムルたちの会話中、物語の分岐に関わる重要な台詞が赤文字で強調される。
プレイヤーはこの赤文字に作用させられるようなテキストを手紙に書く。

また、各場面では背景イラストに存在する物を閲覧することができる。
例えば、机の中にナイフがあることを確認でき、プレイヤーはそれを使うよう指示する旨を手紙に書くことができる。

## 強盗のモンスター
カオティクスの強盗ダンテが来る。
ライカは慌てて屋根裏に隠れる。

ダンテは金品を盗むだけで殺す気はない。
しかし、台所にあった悪夢予防薬の空き瓶を見てネムルがノクタリアンであると気づき、踵を返して殺害に及ぶ。
呆気なく殺害されゲームオーバー。
ニーナがネムルにナイフの場所を知らせておくとネムルは抵抗する。
ナイフは弾き飛ばされ、殺されそうになるが、ライカが屋根裏から飛び出してきてナイフを使ってダンテを殺す。

プレイヤーは序盤時点ではダンテがノクタリアンを憎んでいることや、空き瓶が原因でネムルの種族がバレたことを知らない。
代わりにナイフで抵抗することでダンテを殺害し、先に進むことができる。
物語終盤で悪夢予防薬の瓶が原因であること・ダンテが死ぬと不都合があることをプレイヤーは知る。

- 無警戒→殺される
- 隠れる→お金を取られる
- お金を隠して隠れる→何も起きない
- 抵抗する→殺される
- ナイフで抵抗する→致命傷を与えて

主要な分岐は、
- 殺される 即ゲームオーバー
- 殺す 進行化
- 殺されない ダンテ生還
